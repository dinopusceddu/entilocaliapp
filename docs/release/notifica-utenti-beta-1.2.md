# Notifica Utenti — Rilascio Versione Beta 1.2

**Oggetto: Novità Toolbox Funzioni Locali — Versione Beta 1.2**

Gentili utenti,

siamo lieti di annunciare il rilascio della **Versione Beta 1.2** del Toolbox Funzioni Locali. Questo aggiornamento introduce nuovi strumenti pensati per facilitare la raccolta dei dati iniziali e la gestione documentale del Fondo.

### Cosa cambia con la Beta 1.2?

1. **Nuova barra "Strumenti import/export"**: Abbiamo centralizzato tutti i comandi operativi nella sezione *Dati Generali Ente* per una gestione più intuitiva.
2. **Generatore Lettera Richiesta Dati**: È ora possibile generare automaticamente la lettera formale da inviare all'ente per richiedere i dati mancanti. Il sistema rileva autonomamente quali informazioni mancano e le evidenzia nel testo.
3. **Esportazione Multi-formato**: La lettera può essere scaricata in **PDF professionale**, in formato **Markdown** o copiata direttamente negli appunti.
4. **Importazione Rapida CSV**: Potete caricare i dati anagrafici e tecnici dell'ente tramite un semplice file CSV. È disponibile un **template scaricabile** pronto all'uso.
5. **Chiarezza e Backup**: Abbiamo distinto chiaramente gli strumenti:
    - **CSV**: per la configurazione iniziale guidata dei dati dell'ente.
    - **Excel**: per il backup e il ripristino completo di tutto lo stato del fondo (incluse tutte le risorse e decurtazioni già inserite).

### Sicurezza dei dati
Vi confermiamo che questo aggiornamento **non modifica i dati reali** già inseriti nelle vostre sessioni di lavoro. Le nuove funzionalità sono strumenti aggiuntivi a vostra disposizione.

Il team di sviluppo resta a disposizione per ogni feedback.
